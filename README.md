# GroceryMate – Selenium PyTest Automatisierung

## Projektbeschreibung
Automatisierte End-to-End Tests für den GroceryMate Webshop.
Getestet werden Produktsuche, Warenkorb und Checkout-Prozess, um sicherzustellen, dass Nutzer Produkte zuverlässig finden, auswählen und bestellen können.

## Technologien
- Python
- Selenium WebDriver
- PyTest
- Page Object Model (POM)

## Projektstruktur
tests/ – Enthält die Testskripte  
pages/ – Enthält die Page Object Klassen  
testplan.md – Teststrategie und Vorgehensweise  
testcases.md – Dokumentierte Testfälle  
test_execution_report.md – Testergebnisse

## Installation
pip install -r requirements.txt

## Tests ausführen
pytest -v

## Dokumentation
- Testplan → testplan.md
- Testfälle → testcases.md
- Testergebnisse → test_execution_report.md
